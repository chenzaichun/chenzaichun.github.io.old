用 Python 解方程

2015-08-07
programming
约 207 字 预计阅读 1 分钟

今天无意中发现一张图片，

当如可以通过数学计算网站计算得到结果symbolab

当时病犯了，不承认自己是学渣（貌似我也不是学霸），学过的数学也完全还给了老师，肿么办。只好用 Python 中的SymPy来计算一下了以下方程

{% math_block %} \log_2(x-1) + \log_2x = 1 {% endmath_block %}

安装SymPy

1

pip install sympy

求解，直接上代码

1
2
3
4
5
6

#!/usr/bin/env python

from sympy import *

var('x')
print(solve(log(x-1,2)+log(x,2)-1,x))

结果是2，可是貌似还有一个负数解呢( -1 哪儿去了?)

1
2
3
4
5
6
7
8
9

#!/usr/bin/env python

from sympy import *

var('x')
print(solve(log(x-1,2)+log(x,2)-1,x))

y = Symbol('y', negative=True)
print(solve(log(y-1,2)+log(y,2)-1,y))

暂时只能通过这种方式求解了。

约 191 字 预计阅读 1 分钟

(请确保xcode command line tools已经安装)

直接使用 =pip install= 会出现 =event.h= 找不到：

1

brew install libevent
